XWeb tag library
Tag htmlXmenu

Build HTML content string containing specified menu reading XMenu from DB.

Example: 
		<xweb:htmlXmenu 
			id="testMenu" 
			menuId="/" 
			datasourceName="hibernateXWeb" 
			selectionClass="selected"
			htmlClass="test_class"
			divTop="5"
			ulId="lista_menu"
			inline="true"
			expandFirst="true"
			/>
