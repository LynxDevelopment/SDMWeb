XWeb tag library

Standard Syntax:
     <%@ taglib prefix="xweb" uri="http://www.lynxit.com/xweb/taglib" %>
XML Syntax:
     <anyxmlelement xmlns:xweb="http://www.lynxit.com/xweb/taglib" />
 
The xweb tag library.

Tag Library Information
Display NameXWeb tag library
Version4.1
Short Namexweb
URIhttp://www.lynxit.com/xweb/taglib
 

Tag Summary
chooseThis tag can contain any number of conditional tags, they will be evaluated in order as they appear in an exclusive "when*, else" fashion.
elseThis is a conditional tag that will evaluate its body if none of the preceding conditional tags in the same "choose" tag did.
isInstanceThis is a conditional tag that will evaluate its body if the value of the object attribute is an instance of the class specified by the type attribute.
isSessionValidisSessionValid Tag is used to determine if HttpServletRequest session is valid. Includes the body of the tag if session is valid. The body of the tag is included if issessionvalid matches the value.
isSecureCheck if the HTTP request is a secure connection (https://...)
isSessionFromCookieCheck if the session is maintained by cookie
isSessionFromURLCheck if the session is maintained by URL encoding
cacheLets you cache fragments of your JSP pages
getXmlCreate an xml string with a data of database
applyXslCreate an HTML string thru XSL stylesheet
xmenuBuilds XML string mapping specified menu reading XMenu from DB. it's possible to feed up many (optional) params driving menu tree appearance and / or behaviour.
htmlXmenuBuild HTML content string containing specified menu reading XMenu from DB.
xmenuadminBuild the specified menu reading the menu voices from TB_MENU_ITEMS table.
administratorMenuCreate an xml string with a data of database for the admin from module table
profilerCalculate working time of nested tag
toolbarCreate an xml string representing a navigator toolbar
userAttributeReturns the specified attribute for the User Object stored in the HttpSession
isUserLoggedEvaluates the body if the user is logged in, can be used in xweb:choose
isUserGroupEvaluates the body if the user belongs to the specified group(s), can be used in xweb:choose; 'groupName' attribute can handle one or more group names, separed by comma ','.
hasUserFunctionGives the specified attribute
displayErrorIf empty displays error message and error code passed to the tag, otherwise it puts "errorCode" and "errMsg" attributes in the page context available for it's body only.
getParamReturn the parameter "name" from Http Request
showSourceReturn this source code
generalTrackingStatsgets statistical information about session tracking
monthVisitsStatsgets information about visits in month
isWFStatusEvaluate body tag if the content is in the wfStatus required
mergeXmlCreate an xml string as result of merge of different xml in the tag's body
getCacheTaggives cache's XML string version
getCacheDetailgives verbose cache's XML string version
includeImports a url into the page, has attributes to specify a proxy hostname, port, login and password
boxletImports a url into the page, this url can be defined with the tag configuration or a boxlet descriptor
popupcreates a popup window
useSimulatedSessionInside this tag a simulated session will be used
useRealSessionInside this tag the real session will be used if simulation mode is on
isSimulatedSessionConditional tag that evaluates its body if simulation mode is on
messagesForUserNo Description
exportcreates links to action for export dataset to excel file
exportColumndataset column property
setBundleretrieves a bundle by name
currentMenuItemThis tag retrieves the currently expanded menuItem for the menu state bean with the id specified in the 'id' attribute and sets it into a page context attribute with the name specified in the 'var' attribute.
filterPollsNo Description
contentSearchNo Description
