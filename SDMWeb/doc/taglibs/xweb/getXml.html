XWeb tag library
Tag getXml

Create an xml string with a data of database

Example: 
			<xweb:getXml 
				xmlSource="xwebConfig"
			 	sqlString="SELECT BROWSER_NAME AS BROWSER, COUNT(*) AS USAGE 
						 FROM TB_TRACKING
						 WHERE BROWSER_NAME IS NOT NULL
						 GROUP BY BROWSER_NAME"
				encoding="ISO-8859-1"
				moreparameter="testtesttest"
			 />
