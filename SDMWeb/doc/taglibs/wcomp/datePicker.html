Wcomp tag library
Tag datePicker

Maps a date input field; all attributes must fit data pattern (default is 'dd/MM/yyyy HH:mm')

Example: 
		<wcomp:datePicker 
			id="messageDate" 
			value="${diary.messageDate}" 
			pattern="dd/MM/yyyy HH:mm" 
			clearButton="false" 
			required="true" 
			readOnly="true"/>
