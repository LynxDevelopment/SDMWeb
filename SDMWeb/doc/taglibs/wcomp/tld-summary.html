Wcomp tag library

Standard Syntax:
     <%@ taglib prefix="wcomp" uri="http://www.lynxit.com/wcomp/taglib" %>
XML Syntax:
     <anyxmlelement xmlns:wcomp="http://www.lynxit.com/wcomp/taglib" />
 
The Web UI Components tag library.

Tag Library Information
Display NameWcomp tag library
Version4.1
Short Namewcomp
URIhttp://www.lynxit.com/wcomp/taglib
 

Tag Summary
importComponentsCopy the reference to state beans in the page context
removeComponentRemoves the state beans of the specified component from session
setPropertyPrints out the complete URL to call the binding controller given a set of properties with their values. It is possible to specify a single property to set by the property and value attributes, or to specify more than one property through the com.lynxit.webcomp.commands.SetPropertyTag nested tag. It is also possible to set a URL (different from the default referrer URL) to redirect to after the properties have been set.
setPropertiesPrints out the complete url to call the binding controller given a set of properties with their values. It is possible to specify the properties through the property tag nested tag.
propertySpecify a name-value pair for a component's property
displayDisplays a property value.
aA LinkTag tag prints a html hyperlink to set the specified properties, it works quite likely its parent class com.lynxit.webcomp.commands.SetComponentsPropertiesTag but also prints the html tag and evaluates its body as the body of this html tag. It is possible to set the class and style attributes of the hyperlink via the htmlClass and styletag's attributes. It can be use with ajax, setting the ajax attribute to true; in this case another attribute can be used: usePlaceholder, whose default value is true. When usePlaceholder attribute is enabled, a particular #PLACEHOLDER character is used as a placeholder; it means that the last occurence of this character is searched in the value of onClick attribute and replaced by the redirect url.
editPropertyallows page to display some input field bind to specific property value update.
containersets up a page container
formmaps HTML form object; allows stateless / stateful settings.
clearFormClears a form errors, values or both
inputmaps HTML inputfield oblect; allows stateless / stateful behaviour
fileUpload the file in the specified uploadDir. The uploadDir can be relative to the webapp context (es. ...\webapp[uploadDir]) or absolute (es. uploadDir = C:\... )
intInputspecific input field for integer numbers
doubleInputCreates an input tag (type="text") that accepts only numbers (possibly with decimals). if validation errors occurs an error is set into the State Bean associated to this tag.
hiddenInputCreates an input tag (type="hidden") that accepts any object as its value
textareaCreates a textarea tag
selectDatasetRowPrints a drop-down list that presents the values of the field indicated by the labelField attribute in the dataset indicated by the dataset attribute. Changing the list selection causes the related current row of the dataset to be selected. The attributes htmlClass and style sets the omonimous attributes of the html
labelTag that renders a label UI component
datasetThis tag builds a DataSet object and puts it into the page context for using it in the rest of the page. The tag does not print anything. The attributes of this tag represents (as usual for WebUIComponents views) the default values for its StateBean's properties, they don't have any effect if the StateBean already exists. An exception to this rule is represented by the *query* attribute since it affects the StateBean at any time. This allows to build parametric queries without calling the WebUIComponents controller at any time.
datasourceConnectionDEPRECATED: since version 4.3 datasources instances are automatically managed at request-level. Opens and closes a hibernate session (or a JDBC connection) in which hibernate operations can be performed.
queryParamSets a parameter in the parent dataset tag
datasetFilterallows to set up data filters for dataset elements.
filterConditionunder a parent datasetFilter tag sets up condition for filtering;
isNullA dataset filter that set a "is null" or "is not null" condition on a field
datatableNo Description
rowMust be used inside the body of a 'dataTable' tag, it prints the 'tr' html tag and evaluates its body for each query row. The content of this tag's body can refer to the ${row} and ${rowNumber} values
headerNo Description
ascEvaluates its body if the parent header tag is ordered ascending
descEvaluates its body if the parent header tag is ordered descending
printClobPrints the content of a java.sql.Clob to the jsp output stream
nextPagemaps and handles 'next page' link in a navigation structure
previousPagemaps and handles 'previous page' link in a navigation structure
firstPagemaps and handles 'first page' link in a navigation structure
lastPagemaps and handles 'last page' link in a navigation structure
selecttag maps HTML
optiontag maps HTML
radiomaps the HTML radio input field
checkboxNo Description
enabledThis tag evaluates its body if the outside tag's method isContitionTrue() returns true (this tag must be nested in a tag that implements the ConditionalTag interface)
disabledThis tag evaluates its body if the outside tag's method isConditionTrue() returns false (this tag must be nested in a tag that implements the ConditionalTag interface)
divtag maps it's relative HTML tag;
sumreturn the sum of all "column" values thru the table (data set)
datePickerMaps a date input field; all attributes must fit data pattern (default is 'dd/MM/yyyy HH:mm')
fileBrowserNo Description
editoroutputs a rich text editor in page, eventually pre-filled with text data from page / dataset / etc
