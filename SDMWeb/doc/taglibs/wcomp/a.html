Wcomp tag library
Tag a

A LinkTag tag prints a html hyperlink to set the specified properties, it works quite likely its parent class com.lynxit.webcomp.commands.SetComponentsPropertiesTag but also prints the <a> html tag and evaluates its body as the body of this html tag. It is possible to set the class and style attributes of the hyperlink via the htmlClass and styletag's attributes. It can be use with ajax, setting the ajax attribute to true; in this case another attribute can be used: usePlaceholder, whose default value is true. When usePlaceholder attribute is enabled, a particular #PLACEHOLDER character is used as a placeholder; it means that the last occurence of this character is searched in the value of onClick attribute and replaced by the redirect url.

Example: 
				<wcomp:a
					property="test_prop"
					usePlaceholder="false"
					ajax="false"
					onClick="javascript:document.refresh();"
				><%-- some content --%>
				</wcomps:a>
