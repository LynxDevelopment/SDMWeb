Wcomp tag library
Tag dataset

This tag builds a DataSet object and puts it into the page context for using it in the rest of the page. The tag does not print anything. The attributes of this tag represents (as usual for WebUIComponents views) the default values for its StateBean's properties, they don't have any effect if the StateBean already exists. An exception to this rule is represented by the *query* attribute since it affects the StateBean at any time. This allows to build parametric queries without calling the WebUIComponents controller at any time.

Example: 
			<wcomp:dataset 
				id="boxletconfig_detail"
				query="from Boxlet boxlet where boxlet.id.name = ?"
				dataSource="hibernateXWeb"
				stateful="false">
				<wcomp:queryParam value="${param.BOXLET_NAME}"/>
			</wcomp:dataset>
