Tag setProperty

Prints out the complete URL to call the binding controller given a set of properties with their values. It is possible to specify a single property to set by the property and value attributes, or to specify more than one property through the com.lynxit.webcomp.commands.SetPropertyTag nested tag. It is also possible to set a URL (different from the default referrer URL) to redirect to after the properties have been set.

Example: 
				<wcomp:setProperty
					id="test_property"
					property="test_prop"
					value="test_value"
					ajax="false"
				/>
