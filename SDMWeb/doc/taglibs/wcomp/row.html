Wcomp tag library
Tag row

Must be used inside the body of a 'dataTable' tag, it prints the 'tr' html tag and evaluates its body for each query row. The content of this tag's body can refer to the ${row} and ${rowNumber} values

Example: 
			<wcomp:row omitTrTag="true">
				<%-- put some JSP code here... --%>
			</wcomp:row>
